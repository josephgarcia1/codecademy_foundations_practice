HTML Tags

An HTML tag is commonly defined as a set of characters constituting a formatted command for a Web page. At the core of HTML, tags provide the directions or recipes for the visual content that one sees on the Web.

  • <li> List Item Element
  • <video> Video Element
  • <em> Emphasis Element
  • <ol> Ordered List Element
  • <div> Div Element
  • <br> Line Break Element
  • <img> Image Element
  • <h1>-<h6> Heading Elements
  • <p> Paragraph Element
  • <ul> Unordered List Element
  • <body> Body Element
  • <span> Span Element
  • <strong> Strong Element
  • <a> Anchor Element
  • <head> Head Element
  • <target> Target Attribute
  • <html> HTML Element
  • <title> Title Element
  • <tr> Table Row Element
  • <td> Table Data Element
  • <thead> Table Head Element
  • <tbody> Table Body Element
  • <th> Table Heading Element
  • <tfoot> Table Footer Element
  • <table> Table Element
  • <figure> and
  • <section> and
  • <aside> Aside Element

CSS Selectors

A CSS selector is the first part of a CSS Rule. It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them.

Here are some used on this page:

  • body
  • nav
  • nav ul
  • nav ul a, nav ul
  • header
  • header h1
  • #logo

CSS Properties

A CSS property is a characteristic (like color) whose associated value defines one aspect of how the browser should display the element.

Here are the properties used on this page

body {
margin: 0;
font-family: monospace;
}
nav {
background-color: #282A35;
position: sticky;
top: 0;
text-align: right;
padding-top: 10px;
}
nav ul {
margin-top: 0;
list-style: none;
text-transform: uppercase;
}
nav ul a, nav ul{
text-decoration: none;
font-size: 12px;
color: white;
display: inline-block;
padding: 0px 20px;
}
header {
background-image: url("https://2.bp.blogspot.com/-ir_b0D3SkcI/Wsddfkm4F9I/AAAAAAAAKD8/uQMM1-vR2b8o_eQQ1l5r5ZY_97rxZImAgCLcBGAs/s1600/20120101_192613.JPG");
background-repeat: no-repeat;
background-size: cover;
height: 500px;
text-align: center;
}
header h1{
margin-top: 0;
padding-top: 220px;
font-size: 40px;
font-weight: bolder;
}
#logo {

width: 5%;
